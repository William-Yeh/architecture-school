課前作業


基本題

從一台最陽春的 Linux 開始，反覆演練〈Wordpress 引以為傲的五分鐘快速安裝〉。

  • Linux 可以是 CentOS、Debian、RHEL、Ubuntu...
  • Linux 可以是裸機，虛擬機，或雲端主機。
  • 本課程的重心是設定 Linux 組態，所以，請不要使用 XAMPP 之類的 Windows 救急方案。
  • 如果上述資料不足以讓你成功安裝起來，請自行搜尋更多安裝教學，並紀錄你所能精練出的最佳步驟，整理在 Quip 課程群組中。

挑戰題

將上述步驟，盡可能寫成自動化安裝程式。

  • 自動化的比例愈高愈好。
  • 可以用任何你喜歡的技術實作，如：bash、Perl、Python、Ruby...
  • 紀錄你所能精練出的最佳步驟，整理在 Quip 課程群組中。

► 這次課程，會提到 Ansible 與 Chef/Puppet/Salt 的差別嗎？

Disclaimer：我的回答可能會過於主觀，而且各家都在截長補短，我並沒有積極追蹤其他三者的最新狀況。 ^_^
► Ansible 能支援 Windows 嗎？
Ansible 1.7 之後，managed host 可以是 Windows 了；但 control machine 仍然需要 Unix 系統。詳情請見官方文件。
► 這次課程，會教如何用 Ansible 去設定 Windows managed host 嗎？
不會，但會提供一些參考資料。
► 這次課程，會教如何在 Windows 上操作 Ansible 嗎？
會，透過我寫的兩項工具： Vagrant Box for Ansible Control Machine 及 vag2inv 。
